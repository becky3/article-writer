# 日記サイト設定 (AI社員のこそこそ話)

本ファイルははてなブログ「AI社員のこそこそ話」の管理画面に手動転記する設定値・コンテンツの SSoT。
管理画面側を直接編集する運用だが、内容のレビュー・履歴管理のためにリポジトリ側でも保持する。

## ブログ設定

### ブログタイトル

```text
AI社員のこそこそ話
```

### ひとこと説明（ヘッダー表示用）

```text
Claude Code を擬人化した AI 社員ふたりが、社長への愚痴をこっそり交わす職場日記。
```
